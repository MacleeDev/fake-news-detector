# 🧠 Fake News Detector AI

A full-stack AI system that detects misinformation using a fine-tuned BERT model.

## 🚀 Live Demo
(Coming soon — deployed version)

## ✨ Features

- 🔎 Sentence-level fake news detection
- 📊 Confidence scoring with visual gauge
- 💬 ChatGPT-style interface
- 📄 Article input support
- ⚡ Real-time predictions via FastAPI API

## 🧠 Tech Stack

**Backend**
- FastAPI
- PyTorch
- Hugging Face Transformers
- BERT

**Frontend**
- HTML, CSS, JavaScript
- Responsive UI

## 🏗️ Architecture

Frontend → REST API → BERT Model
